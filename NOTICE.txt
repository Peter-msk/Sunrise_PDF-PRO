Sunrise PDF PRO v1.0.0 업데이트 공지

제품: Sunrise PDF PRO
버전: v1.0.0

주요 변경:
- UI 초기 창 크기, 카드 경계, 글꼴 통일 개선
- 공식 아이콘 투명 모서리 및 UI 아이콘 자산 적용
- Hancom PDF 실제 인쇄 단일 경로 유지
- C++ native 선택 가속 모듈과 Python fallback 유지
- 원클릭 빌드 CMD 유지
- AhnLab/V3 오탐방지 정책 유지

업데이트 정책:
- 본 배포본은 사용자가 직접 다운로드·교체하는 수동 업데이트 방식입니다.
- 프로그램은 백그라운드 자동 다운로드, 자동 설치, 자동 교체를 수행하지 않습니다.
- 업데이트 확인 기능은 사용자가 버튼을 눌렀을 때 latest.json을 확인하는 용도입니다.
- 공식 배포 파일의 SHA256을 확인한 뒤 교체하는 것을 권장합니다.
