# Sunrise PDF PRO

> HWP/HWPX 대형 보고서를 고품질 PDF로 변환하고, 페이지·이미지·A3·제본 규칙을 검수한 뒤 병합·책갈피·제출 패키지까지 생성하는 통합 변환 및 품질검수 도구

## 1. 프로그램 소개

Sunrise PDF PRO는 한글 문서 파일을 PDF로 바꾸는 단순 변환기를 넘어, 여러 개의 HWP/HWPX 파일로 구성된 대형 보고서를 안정적으로 변환하고 최종 제출본의 품질을 확인하기 위한 Windows 업무 프로그램입니다.

환경영향평가서, 설계보고서, 조사보고서처럼 파일 수와 페이지 수가 많고 장·절 구조, 논리 페이지 번호, A3 도면, 컬러 페이지, 제본용 빈 페이지, 책갈피가 중요한 문서를 하나의 프로젝트로 관리하는 것을 목표로 합니다.

변환과 검수는 로컬 PC에서 수행하며, 원본 HWP/HWPX를 직접 수정하지 않습니다. 문제가 발견되면 프로그램이 원본을 임의로 고치는 대신 오류가 발생한 파일과 페이지, 예상값, 실제값, 권장 수정 방법을 사용자에게 알려주는 방향을 따릅니다.

## 2. 개발 목적

1. 한컴오피스를 이용한 HWP/HWPX→PDF 변환을 안정적으로 자동화합니다.
2. 수백 개 파일과 수백·수천 페이지의 대형 프로젝트를 중단 없이 처리합니다.
3. 변환 결과의 페이지 수, 이미지 품질, 논리 페이지, 컬러/A3 배치 등을 자동 검수합니다.
4. 완료된 파일은 재사용하고 실패하거나 변경된 파일만 다시 처리합니다.
5. 검증된 개별 PDF를 순서대로 병합하고 계층형 책갈피와 제출용 검증 자료를 생성합니다.
6. 원본 문서와 사용자 데이터를 외부에 전송하지 않는 로컬 처리 도구를 제공합니다.

## 3. 주요 기능

### HWP/HWPX PDF 변환

- 사용자가 선택한 HWP/HWPX 파일과 프로젝트 폴더 처리
- 로컬 한컴 HWP COM과 Hancom PDF 인쇄 경로 기반 변환
- HWP/HWPX 1쪽과 PDF 1쪽의 대응 유지
- 문서별 안정화 프로필 적용
- 대용량 이미지·OLE·외부 연결 자산이 포함된 문서의 위험도 확인
- 이미지 누락 의심 시 제한된 자동 재변환
- 저해상도 또는 실제 이미지 누락에 대한 실패·경고 처리
- 파일별 한글 프로세스 재시작과 메모리 안정화

### 프로젝트 구조 관리

- 여러 HWP/HWPX 파일을 하나의 보고서 프로젝트로 관리
- 파일 순서와 역할: 표지, 목차, 메인 장, 하위 항목, 부록 등 분류
- 파일명에서 장·절 구조 자동 추정
- 사용자가 잘못 분류된 파일을 직접 수정하거나 검사 제외 지정
- 원본 파일 크기·수정 시각·선택적 SHA-256 기록
- 프로젝트 설정과 진행 상태 저장

### 체크포인트와 이어하기

- 파일별 변환 성공 여부와 결과 PDF 경로 저장
- 프로그램 중단 후 완료된 파일을 다시 변환하지 않고 작업 이어하기
- 실패 파일만 재시도
- 원본이 변경된 파일만 재변환
- 수정된 파일 이후의 페이지 연속성만 다시 검증
- SQLite 기반 실행 이력과 단계별 처리 시간 기록

### 논리 페이지 검수

- PDF 물리 페이지와 보고서 꼬리말의 논리 페이지를 분리해 관리
- 파일 내부 논리 페이지 연속성 검사
- 파일 간 시작·종료 페이지 연속성 검사
- 누락, 중복, 역행, 급격한 번호 증가, 판독 불가 탐지
- 메인 장이 홀수 페이지에서 시작하는지 검사
- 같은 장의 하위 항목이 중단 없이 이어지는지 검사
- 의도된 제본용 빈 페이지와 일반 내용 페이지 구분
- 신뢰도가 낮은 판정은 자동 확정하지 않고 사용자 확인 대상으로 표시

### A3·컬러·품질 검수

- A3 페이지 크기 확인
- A3 내용 페이지의 가로 방향 확인
- A3 내용 페이지가 홀수 논리 페이지에 배치되었는지 확인
- 컬러 페이지 목록 생성
- 이미지 품질과 구조 이상 검사
- 페이지 수와 병합 결과 검증
- 문제 파일, 물리 페이지, 논리 페이지, 예상값과 권장 수정 방법 표시

### 병합·책갈피·제출 패키지

- 검증된 개별 PDF를 최종 순서대로 병합
- 입력 파일별 페이지 범위와 병합 계획 기록
- 메인 장과 하위 항목을 반영한 계층형 책갈피 생성
- 논리 페이지·품질검수 보고서 생성
- 컬러/A3 페이지 목록 생성
- HTML, CSV, TXT 등 검수 결과 출력
- 최종 PDF와 관련 산출물의 SHA-256 제출 패키지 생성

## 4. 기본 업무 흐름

```text
프로젝트 생성 또는 폴더 선택
→ 파일 순서·장 구조 확인
→ 변환 전 사전점검
→ HWP/HWPX 개별 PDF 변환
→ 이미지·페이지·A3·컬러 검수
→ 문제 파일 수정 및 해당 파일만 재변환
→ 연속성 재검증
→ 최종 PDF 병합
→ 계층형 책갈피·검수 보고서·SHA-256 패키지 생성
```

## 5. 중요한 운영 원칙

- 원본 HWP/HWPX를 수정하거나 덮어쓰지 않습니다.
- 변환 결과와 보고서는 별도 출력 폴더에 생성합니다.
- Microsoft Print to PDF, LibreOffice, 임의 SaveAs PDF와 같은 대체 경로를 공식 기본 엔진으로 사용하지 않습니다.
- 페이지 오류를 발견해도 원본 목차, 꼬리말, 표 번호에 영향을 줄 수 있는 자동 수정을 기본값으로 사용하지 않습니다.
- 문서 원본과 로그를 외부 서버로 자동 전송하지 않습니다.
- 한컴오피스 버전, Hancom PDF 프린터, Windows 환경에 따라 실제 변환 결과가 달라질 수 있으므로 공식 릴리즈마다 실문서 검증을 수행합니다.

## 6. 개발 방향

- 대형 프로젝트용 사전점검 센터 강화
- 프로젝트 체크포인트, 작업 이어하기, 변경 파일만 재처리하는 증분 파이프라인 고도화
- 논리 페이지 판독의 정확도와 신뢰도 표시 개선
- A3·빈 페이지·제본 규칙 검수 엔진 고도화
- 오류 페이지로 바로 이동하고 원본 파일을 여는 검수 대시보드 강화
- 네이티브 대량 파일 I/O와 병합 인덱스를 통한 처리 속도 향상
- 한컴 프로세스 상태, heartbeat, 출력 PDF 진행 상태를 감시하는 watchdog 안정화
- 최종 제출본 생성 전 필수 오류를 확인하는 검수 게이트 강화
- 계층형 책갈피와 제출 패키지 형식의 확장
- 자동 업데이트의 원자적 교체, 백업, 재실행, 실패 시 복구 안정화
- Windows 실환경과 AhnLab/V3 환경에서의 반복 검증 체계 유지

## 7. 지원 환경과 배포

- 대상 운영체제: Windows 10/11 64비트
- 필수 환경: 실제 HWP/HWPX 변환을 위한 한컴오피스 설치
- 기본 배포 형태: Windows onefile 실행형
- 현재 공개 README 작성 기준: Sunrise PDF PRO v1.9.0
- 공식 배포본의 기능과 지원 한컴오피스 범위는 릴리즈 노트를 함께 확인해야 합니다.

## 프로젝트 공통 원칙

Sunrise Group의 프로그램은 단순한 일회성 스크립트가 아니라 실제 업무에서 반복 사용할 수 있는 Windows 데스크톱 도구를 목표로 합니다.

- 원본 파일과 사용자 데이터를 우선적으로 보호합니다.
- 삭제, 다운로드, 업데이트, 외부 프로그램 실행 등 중요한 동작은 사용자가 인지하고 승인한 경우에만 수행합니다.
- 사용자 문서, 이미지, 프로젝트 데이터와 API 키를 불필요하게 외부로 전송하지 않습니다.
- 장시간 작업은 진행 상태와 현재 처리 단계를 표시하고, 오류 발생 시 사용자용 안내와 진단 로그를 분리합니다.
- 공식 배포본은 제품 정보, SHA-256 무결성 확인, 코드 서명, AhnLab/V3 사전점검 등 Sunrise Group의 배포 기준을 따릅니다.
- 기능 추가보다 기존 기능의 재현성, 복구 가능성, 대용량 처리 안정성, 유지보수성을 우선합니다.
