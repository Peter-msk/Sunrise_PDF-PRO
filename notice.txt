Sunrise PDF PRO v2.0.0 업데이트 안내

주요 변경사항

- Python 업무 로직과 선택형 C++ 런타임 가드를 결합했습니다.
- 결과 PDF 잠금·원자적 교체·프로세스 생성 토큰·출력 안정화 대기를 네이티브 경로로 보강했습니다.
- 프린터 큐와 현재 변환 단계·경과시간을 확인하는 성능 진단을 추가했습니다.
- 네이티브 모듈을 사용할 수 없을 때는 Python 안전 폴백으로 계속 실행합니다.
- 기존 HWP/HWPX 변환·PDF 검수·병합·책갈피·제출 패키지 기능을 유지합니다.

업데이트 방법

- 프로그램의 업데이트 확인 버튼을 누릅니다.
- 새 버전이 있으면 업데이트 파일 내려받기 여부를 선택합니다.
- 다운로드 EXE의 실제 실행 사전검증이 통과되면 프로그램이 트랜잭션 교체를 준비합니다.
- 새 버전 시작이 확인되지 않으면 이전 실행파일이 자동 복원됩니다.
