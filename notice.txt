Sunrise PDF PRO v1.7.4 업데이트 안내

주요 변경사항

- 서로 다른 폴더의 동일 파일명 PDF도 입력 순서대로 정확히 병합하도록 정규화 캐시 충돌을 수정했습니다.
- 모든 입력 PDF의 사전검증이 끝나기 전에는 병합 엔진과 폴백이 시작되지 않아 일부 파일 누락 결과가 성공 처리되지 않습니다.
- 병합 PDF와 제출 ZIP을 임시 .part 파일에 완성·검증한 뒤 원자적으로 교체하여 중단 시 불완전 결과가 남지 않습니다.
- Windows 빌드 의존성은 지원 범위 안에서만 설치하며, 무조건 최신판으로 올리는 동작을 제거했습니다.
- 실제 예제 PDF로 병합·컬러/A3·논리 페이지·QA·책갈피·SHA-256·ZIP을 통합 회귀검증합니다.
- 검수 게이트를 통과한 프로젝트에 메인 장·하위 항목 계층형 PDF 책갈피를 생성합니다.
- 책갈피 제목에 실제 꼬리말 논리 페이지 범위를 표시합니다.
- 병합 PDF 원본은 수정하지 않고 검증 완료 복사본에 책갈피와 문서 메타데이터를 적용합니다.

업데이트 방법

- 프로그램의 업데이트 확인 버튼을 누릅니다.
- 새 버전이 있으면 업데이트 파일 내려받기 여부를 선택합니다.
- 다운로드 EXE의 실제 실행 사전검증이 통과되면 프로그램이 트랜잭션 교체를 준비합니다.
- 새 버전 시작이 확인되지 않으면 이전 실행파일이 자동 복원됩니다.
