Sunrise PDF PRO v1.3.2 업데이트 안내

주요 변경사항

- 사용자용 사용 환경 확인 내용을 정리했습니다.
- 구조·문법·연결성·빌드 게이트를 재검토했습니다.
- 한글 파일명 ZIP 패키징 안정성을 보완했습니다.
- HWP/HWPX 문서를 Hancom PDF 기준으로 안정적으로 PDF 변환합니다.
- 업데이트 확인 후 사용자가 동의하면 새 실행파일로 자동 교체합니다.

업데이트 방법

- 프로그램의 업데이트 확인 버튼을 누릅니다.
- 새 버전이 있으면 업데이트 파일 내려받기 여부를 선택합니다.
- 실행 중인 프로그램을 종료합니다.
- 기존 실행파일을 새 실행파일로 교체합니다.
