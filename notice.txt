Sunrise PDF PRO v1.1.0 업데이트 안내

주요 변경사항

- 초기 화면 크기와 좌측 작업 패널 표시를 개선했습니다.
- 작업 대시보드와 파일 큐의 카드 경계를 더 명확하게 조정했습니다.
- 전체 UI 글꼴과 버튼 구성을 정리했습니다.
- 공식 아이콘과 기능 버튼 아이콘을 적용했습니다.
- HWP/HWPX 문서를 Hancom PDF 기준으로 안정적으로 PDF 변환합니다.
- 업데이트는 새 실행파일을 내려받아 교체하는 수동 방식입니다.

업데이트 방법

- GitHub Release에서 Sunrise_PDF_PRO.exe를 내려받습니다.
- 실행 중인 프로그램을 종료합니다.
- 기존 실행파일을 새 실행파일로 교체합니다.
