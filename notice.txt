Sunrise PDF PRO v1.7.0 업데이트 안내

주요 변경사항

- 검수 게이트를 통과한 프로젝트에 메인 장·하위 항목 계층형 PDF 책갈피를 생성합니다.
- 책갈피 제목에 실제 꼬리말 논리 페이지 범위를 표시합니다.
- 병합 PDF 원본은 수정하지 않고 검증 완료 복사본에 책갈피와 문서 메타데이터를 적용합니다.
- 프로젝트 매니페스트·논리 페이지 검증·통합 QA 보고서를 최종 제출 패키지에 포함합니다.
- 포함 파일별 SHA-256 체크섬과 무결성 JSON/TXT, 검증 ZIP을 생성합니다.
- 미승인 검수 오류가 남으면 책갈피·제출 패키지 생성을 자동 차단합니다.

업데이트 방법

- 프로그램의 업데이트 확인 버튼을 누릅니다.
- 새 버전이 있으면 업데이트 파일 내려받기 여부를 선택합니다.
- 다운로드 EXE의 실제 실행 사전검증이 통과되면 프로그램이 트랜잭션 교체를 준비합니다.
- 새 버전 시작이 확인되지 않으면 이전 실행파일이 자동 복원됩니다.
