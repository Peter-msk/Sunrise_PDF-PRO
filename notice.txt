Sunrise PDF PRO v1.5.3 업데이트 안내

주요 변경사항

- Python DLL과 Microsoft VC 런타임을 EXE에 명시적으로 포함하고 CArchive 누락을 차단합니다.
- 업데이트 EXE를 교체 전에 실제 실행하여 python312.dll과 필수 모듈 로드를 확인합니다.
- 새 버전이 메인 UI 초기화 health marker를 만들기 전까지 이전 EXE를 보존합니다.
- 새 EXE 시작 실패 시 이전 EXE를 자동 복원하고 실패 로그를 남깁니다.
- 코드서명 후 최종 사용자용·개발자용 EXE 동적 runtime smoke를 빌드 게이트에 추가했습니다.

업데이트 방법

- 프로그램의 업데이트 확인 버튼을 누릅니다.
- 새 버전이 있으면 업데이트 파일 내려받기 여부를 선택합니다.
- 다운로드 EXE의 실제 실행 사전검증이 통과되면 프로그램이 트랜잭션 교체를 준비합니다.
- 새 버전 시작이 확인되지 않으면 이전 실행파일이 자동 복원됩니다.
